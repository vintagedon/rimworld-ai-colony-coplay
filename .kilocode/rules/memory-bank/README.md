# Memory Bank — RimWorld AI Colony Co-Play

Agent context files for persistent memory across Claude sessions.

---

## Files

| File | Purpose | Update Frequency |
|------|---------|------------------|
| [brief.md](brief.md) | Project identity (2-3 paragraphs) | Rarely |
| [product.md](product.md) | Why this exists, goals, phased roadmap | When vision evolves |
| [context.md](context.md) | Current state, next steps, decisions | **Every session** |
| [architecture.md](architecture.md) | System design, structure, patterns | When architecture changes |
| [tech.md](tech.md) | Technologies, setup, constraints | When stack changes |
| [tasks.md](tasks.md) | Repetitive workflows, procedures | As workflows emerge |

---

## Usage

**Session start:** Load brief.md and context.md to understand current state.

**Session end:** Update context.md with accomplishments, next steps, and any decisions made.

**Key principle:** Keep context.md current. Stale context is worse than no context.
